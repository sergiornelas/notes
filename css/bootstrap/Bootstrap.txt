CLASES BOOTSTRAP
	class="container"
		-Anytime we use Bootstrap grid, it needs to be inside
		of a container.
		-Mete el contenido en un div, una caja practica
		con esquinas redondeadas.
		-También se utiliza para CENTRAR un cierto contenido.
			Todo lo que se encuentre DENTRO del container.
	class="container-fluid"
		Lo mismo que container, solo que el contenido del
		container no se centra, no spacing
	class="jumbotron"
		Texto para llamar la atención, con un fondo color gris
	class="thumbnail"
		-Comprime el contenido en un GRID!!
		-Almacena el contenido en un container. 
		-Se utiliza en imágenes. Crea un contorno estético
  	class="glyphicon glyphicon-picture"
		-Agrega una imagen de cámara
		-Éstos íconos se interpretan como texto, es decir
		puedes cambiar su color con "color" en css.
		-Si quieres cambiar el tamaño, cambia el
		tamaño de la fuente (font-size)
	class="active"
		Aparece en la página el contenido seleccionado.
		Se utiliza para poner la pestaña "home" encedida.

	BUTTONS

ETIQUETAS HTML
<style>
	Define el estilo, estética para un archivo HTML
	Se declara en el head, hasta ARRIBA
		<head>
		<style>
		h1 {color:red;}
		p {color:blue;}
		</style>
		</head>		
<label>
	Si el usuario da click al texto de label, habilita
	el control.
		<label for="exampleInputEmail1">Email address</label>

		1) Usuario da click en el texto "Email"
		2) La barra input se activa, proporcionando
		control al usuario de la barra, como si le hubiera
		dado click a la barra.
<nav>
	Define una colección de links de navegación
	El encabezado del 90% de las páginas

<i>
	icon
Mobile responsive website:
	<meta name="viewport" content="width=device-width, initial-scale=1">
	Must be inside <head> element, above the <title> tag.
